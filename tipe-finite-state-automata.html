Finite State Automata DFA & NFA

Finite state automata berdasarkan pada pendefnisian kemampuan berubah state-statenya biasa dikelompokkan kedalam dua bagian, yaitu:

DFA (Deterministic Finite Automata)

Automata dimana setiap state memiliki tepat satu transisi ke state berikut untuk setiap simbol input.

Ciri Deterministic Finite Automata :

  • Dari suatu state selalu memiliki tepat satu transisi ke state berikutnya untuk setiap simbol input.

NFA (Non Deterministic Finite Automata)

Automata dimana dari suatu state dapat:

  • tidak memiliki transisi.
  • tepat satu transisi.
  • lebih dari satu transisi ke state berikutnya untuk simbol input tertentu.

Ciri Non Deterministic Finite Automata :

  • Tidak memiliki transisi dari suatu state untuk simbol input tertentu. Contoh:
    1. Dari state q0 tidak memiliki transisi untuk simbol input b
    2. Dari state q2 tidak memiliki transisi untuk simbol input b
    3. Dari state q3 tidak memiliki transisi untuk simbol input a
  • Dari suatu state memiliki lebih dari satu transisi ke state yang lain untuk simbol input tertentu. Contoh:
    1. Dari state q0 memiliki lebih dari satu transisi untuk simbol input a.
